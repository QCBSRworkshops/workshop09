Workshop 9: Multivariate Analyses in R

Chapter 2 Preparing for the workshop

To prepare for this workshop, you must do the following steps:

Download the data required for this workshop:

  • DoubsEnv data
  • DoubsSpe data

Their data can also be retrieved from the ade4 package:

library(ade4)
data(doubs)

spe <- doubs$fish
env <- doubs$env

Alternatively, from the codep package:

library(codep)
data(Doubs)

spe <- Doubs.fish
env <- Doubs.env

Download the RScript containing the coldiss() function:

  • R Script
  • Coldiss R function

You must also use these packages: * ape * ade4 * codep * gclus * vegan * GGally * PlaneGeometry * remotes

list.of.packages <- c("ape", "ade4", "codep", "gclus", "vegan",
    "GGally", "PlaneGeometry", "remotes", "matlib")

new.packages <- list.of.packages[!(list.of.packages %in% installed.packages()[,
    "Package"])]

if (length(new.packages) > 0) {
    install.packages(new.packages, dependencies = TRUE)
    print(paste0("The following package was installed:", new.packages))
} else if (length(new.packages) == 0) {
    print("All packages were already installed previously")
